Configurar URL do Ngrok

Configure a URL do ngrok para permitir que clientes externos recebam notificações sem precisar estar conectados na rede local do servidor.

Exemplo: https://abc123.ngrok.io
Importante: Deixe vazio para usar apenas rede local
Como usar o Ngrok
  1. Baixe o ngrok em ngrok.com
  2. Execute no terminal: ngrok http 5000
  3. Copie a URL gerada (ex: https://abc123.ngrok.io)
  4. Cole no campo ao lado
  5. Salve a configuração
Dica: A URL do ngrok muda a cada reinicialização, então você precisará atualizar esta configuração sempre que reiniciar o ngrok.
