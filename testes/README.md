# Pasta de Testes

Esta pasta contém todos os arquivos de teste do projeto App Senhas.

## Arquivos Python de Teste

- `test_som_senha.py` - Testes relacionados ao som das senhas
- `test_senha_atual.py` - Testes da funcionalidade de senha atual
- `test_http_notifications.py` - Testes de notificações HTTP
- `test_notifications.py` - Testes gerais de notificações
- `test_qr_code.py` - Testes de geração de QR Code
- `test_network.py` - Testes de conectividade de rede

## Arquivos JavaScript de Teste

- `test_cookies.js` - Testes de cookies no navegador
- `test-clicks.js` - Testes de cliques consecutivos

## Como Executar os Testes

Para executar os testes Python, navegue até esta pasta e execute:

```bash
python test_nome_do_arquivo.py
```

Para os testes JavaScript, abra o arquivo no navegador ou execute em um ambiente Node.js.

## Organização

Todos os arquivos de teste foram movidos para esta pasta para manter o projeto organizado e facilitar a manutenção dos testes.
